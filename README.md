# Portfólio Pessoal

Meu portfólio desenvolvido com React, TypeScript e Tailwind CSS — apresentando projetos, habilidades e formação acadêmica.

🔗 **[renatosilvasousa.github.io/portfolio](https://renatosilvasousa.github.io/portfolio)**

---

## Tech Stack

- **React** + **TypeScript**
- **Tailwind CSS v4**
- **Vite**
- Deploy via **GitHub Pages** (GitHub Actions)

## Rodando localmente

```bash
npm install
npm run dev
```

## Deploy

O deploy é automático via GitHub Actions a cada push na branch `main`.
